# Chat Masha (マシャ)

AI Chatbot berbasis karakter **Maria Mikhailovna Kujou (Masha)** dari _Alya Sometimes Hides Her Feelings in Russian_.

## Fitur

- **Advanced Memory System**: Menyimpan konteks chat dan mengingat informasi penting
- **Edit & Delete Messages**: Bisa edit atau hapus pesan seperti WhatsApp
- **Sticker Support**: Library 100+ sticker untuk ekspresif
- **Natural Indonesian Chat**: Ngobrol pake bahasa Indonesia casual yang natural
- **Desktop App**: Aplikasi desktop pakai Electron

## Cara Pakai

### Requirement

- [Node.js](https://nodejs.org/) (v16 atau lebih baru)
- API Key dari [OpenRouter](https://openrouter.ai/)

### Instalasi

```bash
# Clone repository
git clone https://github.com/kushuri12/AI-ChatBot.git
cd AI-ChatBot

# Install dependencies
npm install
```

### Setup

Buat file `.env` di root folder:

```env
OPENROUTER_API_KEY=your_key_here
```

### Jalanin

```bash
npm start
# atau double-click start.bat
```

## Tech Stack

- Frontend: HTML, JavaScript, Tailwind CSS
- Backend: Node.js, Electron
- AI: Gemini 2.0 Flash via OpenRouter
